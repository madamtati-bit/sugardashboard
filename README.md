# sugardashboard
Tentang SugarDashboard:  Portal analytics pabrik gula dengan sinkron data otomatis dari Google Sheet regional. SugarDash merangkum kesiapan giling (maintenance, bahan baku, procurement), simulasi dampak bottle neck task terhadap awal &amp; akhir giling, serta insight berbasis AI untuk operasional dan keuangan.
